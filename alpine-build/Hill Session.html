Hill Session

Workout Details

The hill sessions are going to be one “click” harder than the CO sessions, so zone 2 or 3 heart rate if you know that scale. This is a challenging session for urbanites, since each session is prescribed with the idea of steady, uphill work for the assigned time.

  • “Hill 20” = 20 minutes of uphill work.
  • If you hike back down, you can count that time into your CO total for the week.

Base Phase Guidance

In the base phase, we don’t need to add load. Just get out and do the work.

Build Phase Guidance

Once we move to the build phase, I suggest a 15-25 pound pack for most of these workouts. As you get closer to the actual objective, you can adapt the load more and more to what your end-goal is. If it’s a guided trip up the Grand Teton, you might be safe with no more than 20 pounds. If it’s a self-supported backcountry trip, you might have to move closer to 60 pounds by the end of the cycle. Regardless, the same rules apply: hard, but sustainable.
